# 5. Comment Modifier Votre Disponibilité — C'est Vous le Patron

L'IA consulte votre Google Calendar **en temps réel**. Si un créneau est occupé, elle ne le propose pas.

### Pour bloquer un créneau :
1. Ouvrez votre **Google Calendar**
2. Créez un événement sur le créneau à bloquer
3. Titre : n'importe quoi (ex: "Réunion staff", "Congé", "Déjeuner")
4. L'IA verra le bloc et ne proposera pas ce créneau

### Pour ajouter des jours de fermeture :
1. Créez un événement "Fermé" sur toute la journée
2. L'IA ne proposera aucun RDV ce jour-là

**Vous n'avez pas besoin de m'appeler pour changer vos horaires. Vous le faites vous-même dans Google Calendar.**

# 9. FAQ — Les Questions du Quotidien

**Q : L'IA peut-elle se tromper sur un rendez-vous ?**
R : Oui, rarement (moins de 10% des cas). Vérifiez votre Google Calendar chaque matin pendant 2 minutes. L'IA confirme toujours la date et l'heure avant de raccrocher, mais une vérification humaine élimine le risque restant.

**Q : Que se passe-t-il si le client parle trop vite ou avec un accent fort ?**
R : L'IA demande poliment de répéter. Si elle ne comprend vraiment pas, elle propose de prendre un message ou de transférer à un humain. Vous pouvez écouter la transcription dans le dashboard.

**Q : Puis-je écouter les appels ?**
R : Oui, si vous activez l'enregistrement dans Vapi (option payante ~$0.02/min). Les enregistrements apparaissent dans votre dashboard. **Conseil :** Activez l'enregistrement pendant les 30 premiers jours pour vérifier la qualité.

**Q : L'IA parle-t-elle arabe / anglais / espagnol ?**
R : Oui, selon la configuration initiale. L'IA détecte automatiquement la langue de l'appelant et répond dans la même langue. Si vous voulez ajouter une langue après la livraison, contactez-moi ($150).

**Q : Que se passe-t-il si je suis en congés ?**
R : L'IA continue de répondre. Elle prend les RDV et les envoie dans votre agenda. Vous les verrez à votre retour. Vous pouvez bloquer vos congés dans Google Calendar pour que l'IA ne propose pas de RDV pendant votre absence.

**Q : Puis-je avoir plusieurs numéros AI ?**
R : Oui, mais chaque numéro supplémentaire nécessite un nouveau setup ($2,500). Contactez-moi pour un devis personnalisé.

**Q : Que se passe-t-il si je veux arrêter le service ?**
R : Arrêtez de recharger Vapi et Twilio. Votre numéro sera désactivé après épuisement du crédit. Vous gardez la configuration et pouvez reprendre quand vous voulez. **Pas de frais de résiliation.**

**Q : Mon assistant(e) humain(e) reste-t-il/elle ?**
R : **Absolument.** L'IA est un assistant pour votre assistant(e). Elle gère les appels répétitifs et les heures d'absence. Votre assistant(e) se concentre sur les tâches à valeur ajoutée : accueil physique, gestion des dossiers, relation client premium. **C'est gagnant-gagnant.**
